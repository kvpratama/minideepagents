# Poetry skill

When writing poetry:

1. Prefer concrete sensory imagery over abstractions.
2. Read each line aloud — if it stumbles, rewrite it.
3. End on a turn: a small surprise that recasts what came before.

For haiku specifically: 5/7/5 syllables, one clean image, one moment in time.
